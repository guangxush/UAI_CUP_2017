# UAI-CUP-2017

## Requirement

Python 2.7

Keras 2.x

sklearn

pandas

## Run

UAI-CUP-2017目录下的文件为各种不同的ML/DL模型代码，运行分为两种方式：

- 只训练，不产生提交结果（用于测试代码的正确性和了解模型效果）,例如：

		python embedding_mlp.py train

- 训练且产生测试集结果用于提交，例如：

		python embedding_mlp.py submit
